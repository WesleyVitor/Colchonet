# README



## Livro


  Ruby on rails coloque sua aplicação web nos trilhos - Vinicius Baggio Fuentes


## Objetivo


  Desenvolver uma aplicação completa para adiquirir conhecimento em ruby e ruby on rails

## Requisitos
  

  * ruby '2.7.0'


  * rails '6.0.3'


## Rodar o projeto

  * Criar as tabelas no banco de dados

    - rails db:migrate


  * Rodar o servidor
  
    - rails server
